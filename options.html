项目介绍

一键总结网页内容，支持多轮对话，兼容 DeepSeek、豆包、Gitee AI 等在线模型，以及 Ollama、Docker Desktop AI、KoboldCpp 等本地模型。

  • 支持网页全文总结、选中文本总结和侧边栏多轮追问。
  • 兼容在线模型和本地模型，可按使用场景灵活切换。
  • 设置页内置连接测试、模型刷新与网络诊断能力。

快速上手

  • 先在“本地模型”或“在线模型”页面完成配置并保存。
  • 打开侧边栏后，点击“总结当前页面”开始生成摘要。
  • 如遇到连通性问题，先到“网络诊断”页检查 provider 状态。

反馈入口 / 使用帮助

GitHub Issues：问题反馈与功能建议 Gitee Issues：国内访问反馈入口
  • 首次使用建议先在设置页完成模型配置，再打开侧边栏执行总结。
  • 如遇到连接问题，可先使用“网络诊断”检查本地和在线 provider 可达性。
